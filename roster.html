Fruit + Lollies Roster¶

The roster shows the order, the person and the date that each parent should contribute with fruits and lollies for the players on each round.

roster.roster¶

Fruit Roster Creator

Script for the fruit roster, using SQLite. The DB file is fruit_roster.db.
